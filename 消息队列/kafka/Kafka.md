# Kafka

## 介绍

Kafka是一种分布式流式计算平台，用于构建实时的数据流水线以及流式计算应用，它是水平伸缩的，容错的，机器快速。

## 三大关键能力

+ 使用容错的方式来存储流式记录

+ 发布和订阅流式记录，类似于消息队列或企业消息系统

+ 处理流式记录

## 优势

+ 构建实时的流式计算数据流水线

+ 构建实时的流式计算应用

## 基本概念

+ Kafka是集群式运行

+ Kafka集群分类存储流式记录，这种分类称为主题

+ 每条记录包含 键，值以及时间戳

### 四类核心API

+ 生产者API
+ 消费者API
+ 流式API
+ 连接器API
